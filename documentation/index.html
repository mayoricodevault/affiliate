Installation

Installation is very easy, just follow these steps and you'll be up and running on your server in no time! Installation is very simliar to most php/mysql applications. Upload, add database, and configure the config file.

  • Step 1. Unzip downloaded file
  • Step 2. Upload contents of "upload" folder to your server
  • Step 3. Create a mysql user and database (recommend using mysql wizard in cpanel)
  • Step 4. Using a program like phpmyadmin import the included affiliatepro.sql file into your new database
  • Step 5. Open config.inc.php located in the "auth" folder
  • Step 6. Add in your database username, password and database name
  • Step 7. In the same file edit the line "$INSTALL_FOLDER = 'affiliate-pro';" ONLY if you've gotten tricky on us and changed the folder name. If in root leave it empty.
  • Step 8. If necessary upload your modified config.inc.php file back to your server to save your changes
  • Step 9. Visit http://yourdomain.com/affiliate-pro and login with default account or create a new one (you will need to use default account initially to assign a new admin user).

    • That's it, you should be up and running!

Website Integration

Integration into current php websites is very simple to do. Integration instructions are also included when you login under "Integration and Settings"

  • Step 1. - Include php tracking file on any page you wish to track (ie, home page) 
    include('affiliate-pro/controller/affiliate-tracking.php');
     * File located in affiliate-pro/controller/affiliate-tracking by default.

  • Step 2. Once a sale has been completed (for example a thank you page) include the file to record the sale into Affiliate Pro. 
    $sale_amount = '21.98';
    $product = 'My Product Description';
    include('affiliate-pro/controller/record-sale.php);
     * File located in affiliate-pro/controller/record-sale by default

  • Step 2b. (Alternative) Use the included PayPal IPN (controller/paypal-ipn.php) sample file instead for additional sale confirmation security through PayPal Service. You will most likely need to create a custom PayPal form and include a custom input to pass the affiliate ID. 
    <input type="hidden" name="custom" value="<?php echo $_COOKIE['ap_ref_tracking'];?>">

Recurring Commissions

To make a commission recurring add the two commission lines to your standard integration (step 2) from above

    Recurring commissions are initialized the same way fixed commissions are with the addition of the two recurring variables below: 
    $sale_amount = '21.98';
    $product = 'My Product Description';
    $recurring = 'monthly'; //can be daily, weekly, biweekly, or monthly
    $recurring_fee = '10'; //percentage of original sale
    include('affiliate-pro/controller/record-sale.php);
     Important: Recurring commissions requires you to setup the following cron job to work. 
    0  1  *  *  *  /usr/bin/php -q /home/username/public_html/affiliate-pro/data/recurring.php

WooCommerce Specific Integration

  • Step 1. - Include php tracking file in wp-content/themes/THEMENAME/header.php (best to use full path) 
    include('/home/myusername/public_html/affiliate-pro/auth/affiliate-tracking.php');

  • Step 2. - Find this line in wp-content/plugins/templates/checkout/thankyou.php (should be line 63) 
    <?php do_action( 'woocommerce_thankyou', $order->id );?>
     After it add this: 
    <?php $sale_amount = $order->get_total(); 
     $product = 'WC Order #: '.$order->id; 
     include('/home/myusername/public_html/affiliate-pro/auth/record-sale.php');?>

    That's it! When an order is completed, it will show up in Affiliate Pro (if an affiliate has referred the sale of course).

Core Features

  • Unlimited Affiliates - Boost Sales by letting Affiliates Sale your Products for you!
  • Affiliate and Admin Panels - Let Affiliates view their own stats / Control it with easy from your own admin Panel
  • Traffic Referral Stats - Keep track of Affiliate Referred Traffic (landing page, ISP...)
  • Banner/Logo Sharing - Upload your banners and logos to give your Affiliate quick access to your marketing materials
  • Commission Levels - Set Commission earning % based on Affiliates level of total sales
  • Track Sales / Profits - Manage all affiliate sold products and earnings with simplicity
  • Manage Payouts - Let your affiliates request payouts. Manage these requests with ease from your admin panel
  • Date Filter - Filter any of the Data based on the Period you want (for example the last 30 days)
  • Quick View Dashboard - View Top Affiliate Preformers, recent sales, traffic on one page
  • View by Affiliate - Zone in and view Sales and Traffic Referral data by Individual Affiliate
  • Void Transactions - Void any transactions if you feel it is appropriate (i.e. refund issued to buyer)
  • Balance Management - Tracks Affiliates Balance (amount owed), if transactions are voided their balance is updated
  • Lots of other Great Features! - View Top Affiliate Preformers, recent sales, traffic on one page

Default Account

By default your first user is the demo account user. Username: demo Password: pass You can register your own users and make them an admin through the user management panel (located in the top right dropdown) by changing the select dropdown. Once your new user is ready, go ahead and delete the demo user if you want.

Multiliqual Support

By default the language is set to English. You can use the selector in the top right corner to change languages as well. If you want to change the default language just open auth>startup.php and change line 34 to your two letter language code. If the language file doesn't exist yet, create your own easily by cloning lang>en.php and rename it with the new country code. Go through the file and make the necessary edits. If you wish to add it to the lanaguage selection dropdown for users simply go to assets>comp>top-nav.php and copy line 25. Paste it below and change en to the appropraite new language code you just created. It might sound like a lot, but really you can create another language for Affiliate Pro in just a couple minutes!

Multi-Currency Formatting

By default the currency is set to USD. You can use the selector in the top right corner to change currencies as well. If you want to change the default currency just open auth>db-connect.php and change line 23 to your currency code. If you wish to add currency selection dropdown for users simply go to assets>comp>top-nav.php and copy line 51. Paste it below and change en-US to the appropraite new currency code you just created.
